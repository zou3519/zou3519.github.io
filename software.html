Acquire
This is a computer clone of the old-school board game Acquire. The objective of Acquire is to earn the most money through investing in and developing hotel chains. The AIs run an imperfect information Monte Carlo Tree Search algorithm that simulates random game play to make decisions. Because the AIs are able to develop strategy by only using the rules of the game, this elegant algorithm generalizes to other non-deterministic games with large search spaces.
Made with Scala and JavaFX.
